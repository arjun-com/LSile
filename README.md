
# LSile

LSile is a _program_ used for transferring **securely** files over a **local network**.

## Run Locally

Clone the repo

```bash
  git clone https://github.com/arjun-com/LSile.git
```

Change directory into the downloaded folder

```bash
  cd LSile
```

Run the program

```bash
  go run main.go help
```
